# 🚀 Todo App

A modern **Todo + Kanban + Focus Dashboard** built using **HTML, CSS, and JavaScript**.

This project combines task management, drag-and-drop workflows, and a clean UI inspired by tools like Notion and Trello.

---

## ✨ Features

* ✅ Add, edit, delete tasks
* ☑ Mark tasks as complete
* ⭐ Focus task system
* 📊 Live task statistics
* 🎯 Filters (All / Active / Completed)
* 🧠 Single source of truth state system
* 🧩 Drag & drop task reordering
* 📦 Kanban board (Todo / Doing / Done)
* 🌙 Dark mode (persistent)
* 💎 Premium UI with smooth animations

---

## 🛠 Tech Stack

* HTML5
* CSS3 (Glass UI + Responsive Layout)
* JavaScript (Vanilla)
* LocalStorage (for persistence)
* SortableJS (drag & drop)
